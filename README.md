# Тестовое задание NordCodes (QA Automation)

Автоматизация тестирования игрового API сервиса.

## Стек технологий
* **Java 21**
* **JUnit 5** — фреймворк для тестирования
* **RestAssured** — работа с API (HTTP-клиент)
* **WireMock** — мокирование внешнего сервиса авторизации
* **Allure** — отчетность по прохождению тестов

## Подготовка к запуску
1. Убедитесь, что у вас установлена **Java 17** или выше.
2. Скачайте файл приложения `internal-0.0.1-SNAPSHOT.jar` и поместите его в корень этого проекта.

## Запуск приложения
Перед запуском тестов необходимо поднять тестируемый сервис. Откройте терминал в корне проекта и выполните:
