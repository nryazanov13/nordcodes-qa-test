# Тестовое задание NordCodes (QA Automation)

Проект по автоматизации тестирования игрового API-сервиса. Реализовано полное покрытие единственного эндпоинта позитивными и негативными сценариями (всего 11 тестов).

## Стек технологий
* **Java 21**
* **JUnit 5** — тестовый движок
* **RestAssured** — HTTP-клиент для тестирования API
* **WireMock (In-memory)** — имитация ответов внешнего сервиса
* **Owner** — управление конфигурацией проекта (библиотека для работы с .properties)
* **Java Faker** — генерация динамических тестовых данных (HEX-токены)
* **Lombok** — использование аннотаций для моделей (DTO)
* **Allure** — интерактивная отчетность с детальным логированием

## Архитектурные решения
* **Step-based API Client**: Использование паттерна "Action Step" — логика запросов инкапсулирована в `AppApi` и `MockApi`.
* **Nested Tests**: Группировка тестов по логическим блокам для чистоты Allure-отчета.
* **Separation of Concerns**: Разделение на слои конфигурации (Owner), спецификаций (RestAssured Specs) и моделей данных.

## Подготовка к запуску
1. Убедитесь, что у вас установлена **Java 17** или выше.
2. Поместите файл приложения `internal-0.0.1-SNAPSHOT.jar` в корневую папку проекта.

## Инструкция по запуску (пошагово)

Для запуска тестов вам не нужно открывать программный код. Достаточно выполнить три простых шага в терминале (командной строке). P.S Есди у вас установен мавен, то команды необходимо выполнять через команду mvn из терминала.

### Шаг 1. Запуск тестируемого приложения
Откройте терминал в папке с проектом и выполните команду:
```bash
java -jar -Dsecret=qazWSXedc -Dmock=http://localhost:8888/ internal-0.0.1-SNAPSHOT.jar
```
### Шаг 2. Запуск автоматических тестов
Откройте второе (новое) окно терминала в той же папке проекта и введите одну из команд.
- Для macOS или Linux:
```bash
./mvnw clean test
```
- Для Windows (PowerShell/CMD):
```bash
./mvnw clean test
```
### Шаг 3. Просмотр отчета Allure
В том же (втором) окне терминала введите одну из команд для открытия отчета.
- Для macOS или Linux:
```bash
./mvnw allure:serve
```
- Для Windows (PowerShell/CMD):
```bash
.\mvnw.cmd allure:serve
```
